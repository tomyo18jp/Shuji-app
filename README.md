# 習字教室 課題プリント作成アプリ

習字教室の課題プリントをブラウザで簡単に作成できるWebアプリです。

## 使い方

1. **用紙を選ぶ** — デフォルト用紙（縦書き・横書き）またはPDF・画像を取り込む
2. **テキスト範囲を指定** — 用紙上で四隅ハンドルをドラッグして配置範囲を決める
3. **テキストを入力** — 直接入力または画像・PDF・Excel・Wordから取り込み
4. **編集** — フォント・文字サイズ・行の区切り位置を調整
5. **印刷** — ブラウザの印刷ダイアログから印刷

## 機能

- 縦書き・横書き対応
- 楷書・行書・草書 9種類のフォント（Google Fonts）
- 両端揃え・自動文字サイズ調整
- 行の区切り位置を1文字単位で調整
- 課題ごとに複数保存・読み込み
- スマホ対応（ピンチアウトで拡大）

## 技術

HTML + CSS + JavaScript のみで動作。外部フレームワーク不使用。
